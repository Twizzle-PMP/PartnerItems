# PartnerItems

plugin de partner items pa cualquier modalidad, hcf, practice, survival, lo que sea

**BETA** — si encontras algun bug mandame dm en discord: **twizzle230**

## comandos

| comando | que hace |
|---------|----------|
| `/abilities all` | te da todas las abilities |
| `/abilities view` | menu decorativo pa ver las abilities |

## dependencias

- PocketMine-MP 5.0.0+
- InvMenu

## instalacion

1. bajate el .phar
2. dropealo en `plugins/`
3. instala InvMenu si no lo tenes
4. reinicia

## config

edita `config.yml` pa cambiar cooldowns y mensajes
